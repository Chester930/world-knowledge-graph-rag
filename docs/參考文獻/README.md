## 下載原則


- 只下載**公開合法**的版本（arXiv 預印本、開放取用期刊、作者自存版）。有版權限制、需訂閱或付費的正式出版版本不下載全文，僅記錄書目資訊供引用查證。
- 每次新增文獻，同步更新 `../論文/附錄與參考文獻.md` 的信任分級表，並在此 README 的清單中補上一列。
