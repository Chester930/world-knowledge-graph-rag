## 5.4 評估指標（草稿）

| 指標類別 | 具體指標 | 說明 |
|---|---|---|
| 答案品質 | Exact Match / F1（若有標準答案）、人工評分（正確性、完整性） | |
| 檢索品質 | Precision@K / Recall@K | 針對圖遍歷取回的事實與人工標註的「應該取回的事實」比對 |
| 幻覺率 | 人工標註「答案中無法從取回內容支持的陳述比例」 | 對應 RQ3，也可參考 Ragas 框架的 faithfulness 指標 |
| 效能 | 端到端延遲（p50/p95）、圖遍歷延遲 | 對應 RQ1、RQ4 |
| 可解釋性（質化） | 案例分析：事實鏈是否可回溯到來源文件 | 對應整體論文主張，質化補充量化指標 |
